class Iterator$<$ pair\+\_\+type, node\+\_\+type $>$}

Custom Forward \mbox{\hyperlink{classIterator}{Iterator}} Template class for members of the binary search tree concept. Every instance of the \mbox{\hyperlink{classIterator}{Iterator}} class is a pointer to a Node type. Mainly used to tarverse the tree in order.
